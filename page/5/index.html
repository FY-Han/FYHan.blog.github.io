生活、学习与娱乐致死的反思-2（科研）

  1. 关于科研
    知乎上看到有人引用ieee fellow 孟庆虎教授的话：“读融创是一位科研工作者进阶的最佳路径。”

  2. 关于成瘾与时间管理

  • 为什么不打一天游戏？
    成瘾性小鼠实验里小鼠会沉迷于按释放多巴胺的按键，而不做别的事情。游戏，和短视频，电影，动漫都是一样的。一个人做这些事情可以，但是你会因为没有找到意义和价值，比如2021.5.23早上下雨，很清凉，我头脑清醒，回顾过去看电影打游戏的三天，会感觉很空虚，因为这种快乐不是你追求的东西。你追求的是具有现实意义的快乐，比如陪伴家人，逛街买东西，坚持运动，科研取得成果。

  • 快乐打一天游戏或者看一天动漫很舒服，追求这种生活并非不合理；但一个是身体受不了，正反馈不断加强最终会导致系统崩溃，另一个是理智告诉我还应该有别的事要做。

  • 需要痛苦或者多巴胺缓释的兴趣，如跑步，练字，健身，跳绳，看书。多巴胺释放的东西不能过多过强，因为少量的多巴胺就能激励我不断的去探索与沉迷；如游戏，或者动漫，最多每次1h，每周最多2次，不能让自己陷入其中，因为还有活要干。除非是安排自己快乐放松，但也不能超过2h/天

不应该做上瘾的，入脑的东西，万事万物应该是我来掌控，而不是让别人掌控我。比如王者，王者视频，asoul。王者大致是可以卸了的。

  • 专注：突破最开始的幻象。跑步中的痛苦，会让自己想停下来，正如学习一会儿想摸手机，打哈欠，没精神。但这都是肉体的幻觉其实，因为自己过于渴望高度有趣的东西，因此会受不了没意思的跑步本身，并放大痛苦的感受。但实际上，这些只是迷雾，只要继续坚持，就会发现能够突破这层迷雾，继续奔跑。比如开始跑了一两百米就累了，想停下来，但听着激昂音乐，最终分两次跑了2.8km。

  • 选择：你的决定就是你的生活。只要你摸到手机，你的生活就会多出很多用手机的时间，你的一天就会用很多手机，也就意味着你放弃了学习。你的决定是什么，你的生活就是什么。不可能说你只做一会儿就停下来，必须在工作时间选择该做的事。

    • 成瘾后，你会感觉疲惫，需要放松，而那种事物可以缓解疲惫，比如想摸摸手机，看看内容，不然手感觉酸，累，乏力，实际这都是幻象。而一旦你摸了手机，你就会

  • 坚持：开始立一个事件目标，最后绷不住的适合列一个时间目标。就像开始说跑2km，最后跑不动的时候再坚持几分钟。不停的切换目标，才能有动力。做事情的时候要给自己列出多种目标，共同促进事情完成。

  • 绷住：只有在指定休息的时间休息。除此之外，早上起床不要摸手机看信息；只有在指定的休息时间（比如忙一个大项目很久之后，忙了一周以后）可以满足自己放松的欲望。

  • 预平衡（以毒素肽为例）
    不是必需的，看初始结构是否平衡。包括：

    1. 能量最小化：使得MD时快速达到平衡，也可以补充需要的离子然后min
    2. 蛋白质位置限制性模拟：限制重原子维持蛋白质结构，放开蛋白质分子的位置限制进行模拟
    3. NVT预平衡：先做NVT减小盒子内压力，防NPT时体系崩溃，再做NPT

    gmx获取拓扑—pdb2gmx module

    gmx-5.x
    gmx pdb2gmx -f fws.pdb -o fws.gro -p fws.top -i fws.itp -ff amber99sb-ildn -water tip3p -ignh
    忽略氢，力场amber99sb-ILDN file：pdb，输出gromacs文件，输出top文件，水盒子
    产生结构文件gro，拓扑文件top，位置限制文件itp
    可以不用ff，会问你用哪个力场

      -ignh: 忽略氢原子,因本pdb文件来自NMR，含氢。统一使用GROMOS力场的氢原子命名规则, 以免名称不一致.
  -ff: 指定使用的力场. 用Amber99SB-ILDN力场. 也可以使用其他力场, 根据体系选择.
  -f: 指定预处理的蛋白质结构文件
  -o: 指定新生成的GROMACS文件名(也可以是其它的文件类型, 如pdb)
  -p: 指定新生成的拓扑文件名, 包含所有原子及原子间的相互作用参数
  -water: 指定水模型. 使用TIP3P水模型. 也可以使用如SPC/E.

  gmx pdb2gmx [-f [<.gro/.g96/...>]] [-o [<.gro/.g96/...>]] [-p [<.top>]]
      [-i [<.itp>]] [-n [<.ndx>]] [-q [<.gro/.g96/...>]]
      [-chainsep <enum>] [-merge <enum>] [-ff <string>] [-water <enum>]
      [-[no]inter] [-[no]ss] [-[no]ter] [-[no]lys] [-[no]arg]
      [-[no]asp] [-[no]glu] [-[no]gln] [-[no]his] [-angle <real>]
      [-dist <real>] [-[no]una] [-[no]ignh] [-[no]missing] [-[no]v]
      [-posrefc <real>] [-vsite <enum>] [-[no]heavyh] [-[no]deuterate]
      [-[no]chargegrp] [-[no]cmap] [-[no]renum] [-[no]rtpres]

      -o      [<.gro/.g96/...>]  (conf.gro)  Structure file: gro g96 pdb brk ent esp
      -p      [<.top>]           (topol.top)       Topology file
      -i      [<.itp>]           (posre.itp)         Include file for topology

    建模拟溶剂盒子—editconf module(或许可以删除)

    gmx-5.x
    gmx editconf -f fws.gro -o fws-PB.gro -bt dodecahedron -d 1.2
    -f: 输入蛋白结构

      ; 近邻列表, 相互作用计算参数
